## 功能

- 拖拽导入 PNG/JPG 作为图层
- 图层多选、拖拽移动、方向键微调（Shift 加速）
- 框选区域并按目标尺寸等比缩放（contain）生成新图层
- 固定尺寸一键框选（默认 `128x128`）
- 框选区域实时显示像素尺寸与旋转角度
- 框选后可直接拖动选框
- 拖拽框选四角控制点可任意角度旋转
- 框选时按住 Shift 锁定 `1:1`
- 活动图层支持 8 个控制点缩放（四角 + 四边）
- 缩放图层时按住 Shift 可保持等比
- 鼠标滚轮平移画布（支持纵向/横向）
- 中键拖动画布
- 修饰键 + 滚轮缩放画布（修饰键可配置）
- 顶部/左侧像素标尺
- 选中图层自动横向/纵向排列
- 导出选中图层（未选中时导出全部）
- 快捷键编辑器（录制新按键、持久化、恢复默认）

## 典型工作流

1. 拖入图片。
2. 选中图层后开始框选，或用“一键创建框选”。
3. 输入目标尺寸（如 `128x128`），生成新图层。
4. 多选图层后横向排列。
5. 导出 PNG。

例如 6 个 `128x128` 图层横向排列，导出尺寸为 `768x128`。

## 画布操作

- 滚轮：平移画布
- 中键拖拽：平移画布
- 缩放：`缩放修饰键 + 滚轮`
  - 默认修饰键：`Alt/Option(⌥)`
  - 可在 UI 中切换为 `Ctrl` / `Meta(Command)` / `Shift` / `无`
- 框选模式：拖动四角圆点可自由旋转框选
- 图层变换：拖动活动图层控制点可缩放（`Shift` 保持等比）

## 默认快捷键（可在 UI 中改）

- `C`：开始/结束框选
- `X` 或 `Esc`：清除框选
- `R` 或 `Enter`：从框选生成新图层
- `B`：一键创建固定尺寸框选
- `G`：自动散开图层
- `H`：横向排列
- `V`：纵向排列
- `Delete` 或 `Backspace`：删除选中图层
- `E`：导出 PNG
- `方向键`：移动选中图层（`Shift + 方向键` 为 10px 步进）
